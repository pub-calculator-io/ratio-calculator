The ratio calculator simplifies ratios by bringing ratios to the lowest terms. Finds missing values in proportions and compares two given ratios finding if they are equal.

[https://www.calculator.io/ratio-calculator/](https://www.calculator.io/ratio-calculator/)

== Usage ==

Add the Ratio Calculator shortcode to your page, post or sidebar:

`[ci_ratio_calculator]`

Add the following code to your template where you would like the Ratio Calculator to appear:

`<?php display_ci_ratio_calculator(); ?>`

== Installation ==

1. Upload the Ratio Calculator /ci_ratio_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Ratio Calculator plugin through the "Plugins" menu in WordPress.
